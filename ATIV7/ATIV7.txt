Laboratório - Todos exercícios usarão pelo menos DUAS estruturas de repetição diferentes. 


Exercício 1 - Usar Faça ... Enquanto para não aceitar numeros negativos e Enquanto para encerrar o algoritmo.


Faça um algoritmo que calcule o salário e o imposto de renda de um grupo de pessoas, considerando:
a) Para cada pessoa tem-se: Nome, Número de Dependentes, Renda Bruta Anual;

b) O imposto é calculado segundo a tabela abaixo:

  Renda Liquida  %

  até R$ 10.000,00  isento

  > R$ 10.000,00 até R$ 30.000,00  5%

  > R$ 30.000,00 até R$ 60.000,00   10%

  > R$ 60.000,00  15%

  > não aceitar valores negativos.

c) Há um desconto de R$ 600,00 para cada dependente.

d) Sendo que a renda liquida é igual a renda bruta menos o abatimento de cada dependente.

e) O algoritmo encerra ao se digitar "FIM" para nome.
